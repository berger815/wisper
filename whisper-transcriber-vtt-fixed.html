Transcriber

Local Whisper transcription with timestamps, VTT export, and live dictation.engine: —
WebGPU uses a mixed-precision Whisper build to reduce GPU memory. If model loading succeeds but the first inference fails, the app now abandons WebGPU and retries in single-thread WASM instead of recreating the same failing worker.
Drop an audio file here, or click to choose .m4a .mp3 .wav .ogg .webm — decoded locally in your browser
Speaks segments as you pause (~0.7s of silence flushes a segment). Use tiny.en or base.en for the lowest lag. Screen stays awake while dictating.
.vtt / .srt need timestamped segments (any run in this version produces them). Hand edits in the box apply to .txt only.
Model runs via transformers.js v3.8.1 (WebGPU/WASM) after a one-time download. Model reuse depends on the browser cache and available storage.
