Start Here — Build a Defensible Inventory Policy

This workbook separates calculation from approval. It chooses a model only when inputs support it, exposes assumptions, and prevents invalid rows from quietly entering portfolio totals. v4.0 adds the Stock Chart and Milestones system for time-phased policy planning.

1 — Setup

Record the project, supplier, owner, revision, and assumptions. These fields travel with the JSON record.

2 — Demand

Choose Annual/Quarterly/Monthly; define each year as Historical or Forecast; enter demand. Average, SD, and CV calculate automatically.

3 — Parts & Policy

Set the review policy, lead time, service target, and cost. Review every yellow assumption and red blocking error before using totals.

4 — Stock Chart

Visualise safety stock, cycle stock, and average inventory level overlaid on the demand profile. Milestone markers appear as vertical lines on the chart.

Stock Chart Interpretation
SeriesMeaning
Demand barsPeriod demand in units. Uses the same time-phased data from the Demand tab.
Safety Stock lineThe computed SS floor. This is constant unless modified by a milestone. A post-milestone value overrides the baseline.
Cycle Stock line½ × Order Quantity — the average working stock above the SS floor, assuming EOQ or fixed order quantity.
Avg Inventory lineSS + ½ OQ — the expected average on-hand level.
ROP / Target lineThe reorder point or order-up-to target. Stepped changes driven by milestones are shown as a stepped line.
Milestone markersVertical dashed lines at milestone dates. Tooltip shows milestone details on hover.
